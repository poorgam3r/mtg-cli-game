# Security Policy

## Supported Versions

We support the latest stable version of this project.

## Reporting a Vulnerability

If you discover a security vulnerability in **MTG CLI Game**, please report it responsibly.

**How to report:**

- Email: mtgcligame.class896@passmail.net
- GitHub Issue: [Open a private issue](https://github.com/poorgam3r/mtg-cli-game/issues)

We will review and respond as soon as possible. Please do not disclose security issues publicly until they have been reviewed and resolved.

## Responsible Disclosure

We greatly appreciate responsible disclosures that help keep this project safe for all users. Thank you!
